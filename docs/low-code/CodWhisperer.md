# 단축키
Code Whisperer를 이용하는 환경에 따라서 다를 수 있음.

* Manually trigger CodeWhisperer
  + MacOS: Option + C
  + Windows: Alt + C

* Accept a recommendation
  + Tab

* Next recommendation
  + Down or Right arrow
* Previous recommendation
  + Up or Left arrow

* Reject a recommendation
  + ESC
